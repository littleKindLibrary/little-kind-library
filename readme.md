# وبسایت کتابخانه کوچک مهربانی (Little Kind Library)

این مخزن حاوی کدهای وبسایت جدید «کتابخانه کوچک مهربانی» است. [cite_start]هدف ما ایجاد خانه‌ای دیجیتال است که برای کودکان شاد، برای تسهیلگران سریع و برای تیم فنی پایدار باشد. [cite: 6, 7, 8, 9]

## تکنولوژی‌ها
[cite_start]ما برای چابکی و کاهش هزینه‌ها از معماری Jamstack استفاده می‌کنیم: [cite: 3]
- [cite_start]**موتور سایت‌ساز:** Hugo (بسیار سریع و امن) [cite: 14]
- [cite_start]**قالب پایه:** Hugoplate (مینیمال و مدرن) [cite: 17, 18]
- [cite_start]**سیستم مدیریت محتوا:** Decap CMS / Headless (برای نویسندگان بدون دانش فنی) [cite: 60]

## نحوه مشارکت (Contributing)
ما از مشارکت برنامه‌نویسان و طراحان استقبال می‌کنیم. [cite_start]کدهای این پروژه به صورت Open Source در دسترس است تا همه بتوانند در توسعه آن سهیم باشند. [cite: 47, 50]

### راه‌اندازی روی سیستم شما
1. این مخزن را کلون کنید.
2. دستور `hugo server` را اجرا کنید.
3. سایت در آدرس `localhost:1313` در دسترس خواهد بود.
